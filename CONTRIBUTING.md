# Contributing

## Environment setup

**Requirements**

- Windows
- Python `3.10+`
- [pandoc](https://pandoc.org/installing.html)
- git

1. Clone the repository.

```
git clone https://github.com/wisdomwater/pistis-sophia && cd pistis-sophia
```

2. Run the setup script.

```
do setup
```

## Compile the sources

```
do compile
```
